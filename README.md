# Map-font-symbol
🦜 A set of icons for making maps 

The icons are available as svg and font.

## SJJB

These map icons are an attempt to create a simple consistent set of vector icons suitable for cartographic use.    
CC0 &copy; [SJJJB](https://www.sjjb.co.uk/mapicons/)

## Maki

Maki is an icon set made for map designers. Maki includes icons for common points of interest like parks, museums, and places of worship. 
CC0 &copy; [Mapbox](https://labs.mapbox.com/maki-icons/)


## Pirate, Princess
A set of icons for making halloween maps.   
CC0 &copy; [Viglino](https://github.com/Viglino/map-font-symbols)

## Evilz

A set of icons for making halloween maps.   
Freeware &copy; [N+ Sakurai Nan aka "Bosskete"](https://www.fontspace.com/evilz-font-f4530)
